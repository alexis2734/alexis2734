Hola Computacion

Es la Ciencia que trata del estudio de procesamiento de datos atravez de una computadora


Hardware

Corresponde a todas las partes fisicas y tangibles de una computadora y sus componentes electronicos, electrodomesticos y mecanicos de todos los elementos

Software

Es una serie de programas que permiten la resolucion de problemas utilizando como medio la ejecucion de la computadora.

El software

  • Microsoft Word - Procesador de texto
  • Microsoft Excel - Hojas de cálculo
  • Microsoft PowerPoint - Presentaciones
  • Microsoft Outlook - Correo electrónico y calendario
  • Microsoft Access - Base de datos
  • Microsoft Publisher - Diseño y publicación
  • Microsoft OneNote - Notas y organización
  • Microsoft Teams - Comunicación y colaboración
